# 猜歌王 - Vue版本

这是一个基于Vue 3和Vant UI框架开发的猜歌游戏。

## 技术栈

- Vue 3
- Vite
- Vant 4
- Vue Router

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 预览

```bash
npm run preview
```

## 项目结构

```
├── src/
│   ├── views/          # 页面组件
│   │   ├── Home.vue    # 首页
│   │   └── GuessSong.vue # 猜歌页面
│   ├── router/         # 路由配置
│   ├── App.vue         # 根组件
│   └── main.js         # 入口文件
├── Qustion/            # 题目和音频文件
├── public/             # 静态资源
└── index.html          # HTML模板
```
